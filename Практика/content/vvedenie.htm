ВВЕДЕНИЕ


Данный раздел электронного ресурса по учебной дисциплине «Аппаратное обеспечение интеллектуальных систем» (АОИС) для студентов специальности 1-40-03 01 состоит из двух подразделов:

  1. Лабораторный практикум по дисциплине для дневной формы обучения;

  2. Контрольные и практические работы для студентов дистанционной формы обучения.

Первый подраздел содержит материалы лабораторного практикума по данной дисциплине, предназначенные для облегчения понимания и закрепления знаний по информационным и логическим основам построения ЭВМ, приобретения навыков по построению и проверке программных моделей ассоциативных памятей и ассоциативных процессоров разного типа.

Лабораторный практикум содержит материалы по восьми лаборатрным работам.

Материал по каждой лабораторной работе включает:

  • цель работы;

  • краткие теоретические сведения, необходимые для выполнения работы (эти материалы есть в соответствующих темах теоретической части данного ЭРУД, но для удобства они повторены или дополнены в более кратком виде и в лабораторных работах);

  • контрольные вопросы для самопроверки;

  • индивидуальные задания на выполнение работы;

  • требования к программе;

  • порядок выполнения работы.

При разработке программных моделей могут быть использованы рекомендации, приведенные в лабораторных практикумах [19, 26].

Второй раздел содержит методические материалы к:

  • контрольной работе по разделу 2 дисциплины «Информационно-логические основы построения ЭВМ»;

  • по двум индивидуальным практическим работам по ряду основных тем дисциплины, касающихся синтеза логических схем и моделирования ассоциативной памяти.

Материалы по каждому виду работ второго раздела включает:

  • цель работы;

  • краткие теоретические сведения, необходимые для выполнения работы;

  • примеры выполнения задач по теме работы;

  • контрольные вопросы для самопроверки;

  • индивидуальные задания на выполнение работы;

  • порядок выполнения и защиты работы.
