Л.р.№1 Описание действий и задач в семантической памяти.

Цель работы:

Научиться описывать в семантической памяти при помощи соответствующих языковых средств условия пошагового решения и результата для задач нескольких типов.

Задачи лабораторной работы:

  1. Познакомиться с понятием действия, задачи, классификация действий, классификация задач.
  2. Изучить язык описания действий и задач в семантической памяти.

Варианты заданий:

  1. Описать в семантической памяти условия пошагового решения для действия "Элементом каких множеств является указываемая сущность и какие роли она там выполняет?"
  2. Описать в семантической памяти условия пошагового решения для действия "Какие элементы принадлежат указываемому множеству и какие роли они выполняют?"
  3. Описать в семантической памяти условия пошагового решения для действия "Элементом каких множеств является указываемая сущность?"
  4. Описать в семантической памяти условия пошагового решения для действия "Какие элементы принадлежат указываемому множеству?"
  5. Описать в семантической памяти условия пошагового решения для действия "Какие варианты декомпозиции соответствуют указываемой сущности?"
  6. Описать в семантической памяти условия пошагового решения для действия "Какие сущности являются частными по отношению к указываемой сущности?"
  7. Описать в семантической памяти условия пошагового решения для действия "Какие сущности являются общими по отношению к указываемой сущности?"
