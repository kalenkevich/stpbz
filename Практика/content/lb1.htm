Л.р.№1 Работа в командной строке. Системы контроля версий.



Цель работы:

Освоить командные оболочки shell (для OС семейства Unix) и cmd (для OС семейства MS Windows):

  • изучить основные встроенные команды,

  • научиться писать файлы сценариев,

  • научиться соотносить командные оболочки для разных OS.

Освоить командное окружение для OС семейства Unix(утилиты из пакета GNU Core Utilities), и соответствующие им утилиты для OС семейства MS Windows.

Научиться работать с системой контроля версий Git.



Задачи лабораторной работы:

  1. Познакомиться с консольными программными окружениями OС Windows и OС Linux. Получить практические навыки работы в консоли, использования консольных команд.

  2. Познакомиться с программами git и gitk, освоить их программные аргументы. Создать текстовый документ с использованием git. Выполнить последовательность шагов
