Решение задач по синтезу комбинационных схем (КС), т.е. схем, не содержащих элементов памяти, выполняется обычно в следующей последовательности:

  1. Описание поведения КС, определение количества входов (логических переменных) и выходов (функций).

  2. Составление таблицы истинности, описывающей поведение КС.

  3. Составление математических выражений (логических функций) для выходных сигналов в заданной форме – совершенной дизъюнктивной нормальной форме (СДНФ) или совершенной конъюнктивной нормальной форме (СКНФ).

  4. Минимизация логических функций выбранным методом (расчетным, расчетно-табличным или табличным).

  5. Составление логической схемы КС на основании минимизированных логических функций.

В теоретической части ЭУМКД по дисциплине АОИС были рассмотрены вопросы синтеза КС следующих типов:

  1. Синтез двоичного полусумматора на 2 входа и двоичного одноразрядного сумматора на 3 входа (ОДС-3), с раздельной минимизацией логических функций.

  2. Алгоритм синтеза логических устройств с несколькими выходами, основанный на возможности выражать одни логические функции через другие.

  3. Алгоритм синтеза логических устройств с не полностью определенными функциями.

Синтез указанных выше типов КС производился с использованием СДНФ для представления выходных функций. Но для синтеза КС может использоваться и СКНФ. При этом синтез КС выполняется в рассмотренной выше последовательности, но с учетом особенностей представления функции в СКНФ на основании таблицы истинности.

Отметим, что функция bi+1 является как бы частью функции di, если бы не было коституэнты k4=(1i+x2i+x3i), т.е. частью функции di является некоторая другая функция bi*, равная bi+1 без конституэнты k4.

b*i+1(x1i, x2i, x3i)=(x1i+x2i+x3i)(1i+x2i+3i)(1i+2i+x3i) (5)

что условно можно записать следующим образом:

b*i+1= bi+1/(1i+x2i+x3i), (6)

Условно, так как в алгебре логики нет функции деления, но это можно заменить другими функциям, например, за счет сложения функции bi+1 с конституэнтой 1, равной (x1i2i3i).

Действительно эта конституэнта в 2n-1 случаях из 2n равна 0. В случаях, когда эта конституэнта равна 1, как раз и обеспечивается значение b*i+1=1 на наборе переменных x1i2i3i=1.

Следовательно, сложение функций bi с логическим произведением (x1i2i3i) обеспечивает полную нейтрализацию ненужной конституэнты k4=(1i+x2i+x3i), что эквивалентно делению bi на эту конституэнту (k4) тогда:

b*i+1= bi+1 + x1i2i3i ­­ (7)

di = [bi+1+ (x1i2i3i)]· (1i+x2i+x3i) (8)

Это наиболее удобно производить при помощи таблицы Вейча-Карно.

- произвести запись значений ЛФ (как 1, так и 0) для всех неизбыточных наборов аргументов в таблицу Вейча-Карно;

- доопределить ЛФ по таблице либо 1, либо 0 так, чтобы отмеченные клетки составляли по возможности наиболее крупные совокупности прямоугольников по 2 клеток. Необходимо, чтобы в отмеченные конфигурации обязательно входила хотя бы одна реализация функции, соответствующая неизбыточному набору;

- по таблице Вейча-Карно произвести запись тупиковой ДНФ или КНФ доопределенной ЛФ обычными способами.
