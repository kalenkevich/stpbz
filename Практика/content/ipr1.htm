Синтез комбинационных схем.

Решение задач по синтезу комбинационных схем (КС), т.е. схем, не содержащих элементов памяти, выполняется обычно в следующей последовательности:

  1. Описание поведения КС, определение количества входов (логических переменных) и выходов (функций).

  2. Составление таблицы истинности, описывающей поведение КС.

  3. Составление математических выражений (логических функций) для выходных сигналов в заданной форме – совершенной дизъюнктивной нормальной форме (СДНФ) или совершенной конъюктивной нормальной форме (СКНФ).

  4. Минимизация логических функций выбранным методом (расчетным, расчетно-табличным или табличным).

  5. Составление логической схемы КС на основании минимизированных логических функций.

В теоретической части ЭУМКД по дисциплине АОИС были рассмотрены вопросы синтеза КС следующих типов:

  1. Синтез двоичного полусумматора на 2 входа и двоичного одно разрядного сумматора на 3 входа (ОДС-3), с раздельной минимизацией логических функций.

  2. Алгоритм синтеза логических устройств с нескольким выходами основанной на возможности выражать одни логические функции через другие.

  3. Алгоритм синтеза логических устройств с не полностью определёнными функциями.

Синтез указанных выше типа КС производился с использованием СДНФ для представления выходных функций. Но для синтеза КС может использоваться и СКНФ. При этом синтез КС выполняется в рассмотренной выше последовательности, но с учетом особенности представления функции в СКНФ на основании таблица истинности.

b*i+1= bi+1 + x1i2i3i ­­ (7)

di = [bi+1+ (x1i2i3i)]· (1i+x2i+x3i) (8)

- произвести запись значений ЛФ (как 1, так и 0) для всех не избыточных наборов аргументов в таблицу Вейча-Карно;

- доопределить ЛФ по таблице либо 1, либо 0 так, чтобы отмеченные клетки составляли по возможности наиболее крупные совокупности прямоугольников по 2 клеток. Необходимо, чтобы в отмеченные конфигурации обязательно входила хотя бы одна реализация функции, соответствующая не избыточному набору;
